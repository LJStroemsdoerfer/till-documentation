    Getting Started¶

    Installation¶

    till can be downloaded from PyPi. We provide a standard Python interface, so no surpises there. 

    $ pip install till

    Once you downloaded till, you will need to activate your license using the built-in method till.activate_license(). Simply run this command and you will be able to use our Python API. Make sure are connected to the internet while doing so.
